# Thwomp

Drum synthesizer.

![Thwomp](Thwomp.png)

## Description

Thwomp is a four oscillator drum synthesizer.

## Presets

### 1. Kick Drum

#### Body

##### Oscillator 1

- `65Hz` Cycle (Sine) Wave
- `1ms` Attack
- `383ms` Decay

##### Pitch Envelope

- `120Hz` Amount
- `100ms` Decay

#### Transient/Click
