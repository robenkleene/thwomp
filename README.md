# Thwomp

Drum synthesizer.

![Thwomp](Thwomp.png)

## Description

Thwomp is a four oscillator drum synthesizer.

## Presets

### 1. Kick Drum

#### Oscillator 1 (Body)

Main body.

- `65Hz` Cycle
- `1ms` Attack
- `383ms` Decay

##### Pitch Envelope

- `120Hz` Amount
- `100ms` Decay

#### Oscillator 2 (Transient/Click)

Main click.

- `251Hz` Rect
- `0ms` Attack
- `26ms` Decay

##### Filter

- `bandpass` Type
- `622Hz`
- `9.1dB`
- `5.6Q` 

#### Oscillator 3 (Body)

Subtle sound to flesh out the body.

- `120Hz` Tri
- `41ms` Attack
- `262ms` Decay

##### Pitch Envelope

- `66Hz` Amount
- `170ms` Decay

##### Filter

- `lowpass` Type
- `240Hz`
- `0dB`
- `1Q`

#### Oscillator 4 (Transient/Click)

Subtle sound to flesh out the click.

- `65Hz` Tri
- `41ms` Attack
- `262ms` Decay
